# H264ToMP4 发布说明 v1.0.0

## 发布概述

H264ToMP4 v1.0.0 是该工具的第一个正式发布版本，提供了将海雀监控摄像头录制的.v264格式视频文件转换为标准MP4格式的完整功能。

## 主要功能

### 核心功能
- 批量转码.v264格式视频文件为标准MP4格式
- 支持多线程并行处理，提高转码效率
- 提供直观的图形用户界面，操作简单
- 支持视频片段合并功能
- 可选择是否处理音频流

### 高级功能
- 实时进度监控和状态显示
- 支持暂停、继续、取消转码操作
- 失败任务重试机制
- 详细的日志记录和错误处理
- 用户配置自动保存

## 系统要求

### 最低要求
- 操作系统：Windows 7 (64位)
- 内存：4GB RAM
- 处理器：双核 2.0GHz CPU
- 硬盘空间：至少1GB可用空间

### 推荐配置
- 操作系统：Windows 10/11 (64位)
- 内存：8GB RAM 或更高
- 处理器：四核 2.5GHz CPU 或更高
- 硬盘空间：至少10GB可用空间

## 安装指南

### 便携版安装
1. 下载 H264ToMP4_v1.0.0_Portable.zip
2. 解压到任意目录（建议使用英文路径）
3. 双击运行 H264ToMP4.exe

### 安装版安装
1. 下载 H264ToMP4_v1.0.0_Setup.exe
2. 运行安装程序，按照提示完成安装
3. 从开始菜单或桌面快捷方式启动程序

## 使用指南

### 基本使用步骤
1. 启动程序
2. 点击"浏览"按钮选择包含.v264文件的源目录
3. 选择输出目录（可选，默认为源目录下的converted子目录）
4. 点击"扫描目录"按钮，程序会自动列出所有可转换的视频文件
5. 根据需要配置选项：
   - 勾选"合并为一个视频"可将所有片段合并为一个文件
   - 勾选"包含音频处理"可处理视频中的音频（如果存在）
6. 点击"开始转码"按钮开始处理
7. 在界面中监控转码进度和状态
8. 转码完成后，在输出目录中找到转换后的MP4文件

### 高级操作
- **暂停转码**：点击"暂停"按钮可临时停止转码
- **继续转码**：点击"继续"按钮可从暂停处恢复转码
- **取消转码**：点击"取消"按钮可停止所有正在进行的转码任务
- **重试失败任务**：转码完成后，可点击"重试失败任务"按钮重新处理失败的文件

## 配置说明

程序会自动创建config.json配置文件，保存用户偏好设置。主要配置项包括：

- `source_dir`：源文件目录
- `output_dir`：输出文件目录
- `ffmpeg_path`：FFmpeg可执行文件路径
- `video_codec`：视频编码器（默认libx264）
- `crf`：视频质量参数（默认18，数值越小质量越高）
- `audio_codec`：音频编码器（默认aac）
- `audio_bitrate`：音频比特率（默认128k）
- `threads`：转码线程数（默认4）
- `overwrite`：是否覆盖已存在的文件（默认false）
- `keep_original`：是否保留原始文件（默认true）
- `log_level`：日志级别（默认INFO）

## 已知问题

1. 处理大量小文件时可能需要较长时间
2. 转码过程中CPU占用率较高
3. 某些特殊格式的.v264文件可能无法正确识别
4. 在某些系统上可能需要管理员权限运行

## 故障排除

### 转码失败
- 检查FFmpeg是否正确安装
- 检查输入文件是否损坏
- 查看日志文件获取详细错误信息

### 程序无法启动
- 确保系统满足最低要求
- 检查是否有防病毒软件阻止程序运行
- 尝试以管理员身份运行

### 转码速度慢
- 尝试增加配置文件中的threads数值
- 确保硬盘有足够的读写速度
- 关闭其他占用CPU资源的程序
