# 更新日志

## [1.0.0] - 2024-01-01

### 新增
- 初始版本发布
- 支持海雀监控.v264格式视频文件批量转码为MP4格式
- 提供图形用户界面，支持文件夹选择和批量处理
- 支持多线程并行转码，提高处理效率
- 支持视频合并功能，可将多个视频片段合并为一个文件
- 支持音频处理选项，可选择是否保留并转码音频
- 提供详细的进度显示和日志记录
- 支持暂停、继续和取消转码操作
- 支持失败任务重试功能
- 自动生成标准MP4格式，支持主流播放器播放和拖动进度条

### 技术特点
- 使用FFmpeg进行视频转码，确保高质量输出
- 采用模块化设计，代码结构清晰
- 支持配置文件保存用户偏好设置
- 完善的错误处理和日志记录机制
- 支持打包为独立可执行文件，无需安装Python环境

### 系统要求
- Windows 7/8/10/11 (64位)
- 4GB以上内存
- 足够的磁盘空间存储转码后的视频文件

### 依赖项
- FFmpeg (已包含在发布包中)
- Python 3.7+ (仅开发环境需要)
- tkinter (通常随Python安装)

### 已知问题
- 处理大量小文件时可能需要较长时间
- 转码过程中CPU占用率较高
- 某些特殊格式的.v264文件可能无法正确识别

### 未来计划
- 添加更多视频格式支持
- 优化转码性能
- 添加视频预览功能
- 支持更多视频参数调整
- 添加命令行界面支持
